## General info
A simple application to interact with a machine learning model. Allows recognition of the breed of dog.

You can test the project [here](https://dog-breed-recognition.kamilbil.pl/).

## Technologies
The project is created with:
* Angular
* TypeScript
* Docker
* Ninx
* PrimeNg
